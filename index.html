Web Component Manifest Graph

Dependency graph generator for WCM

Package description

A complex tree of dependencies underpins most single-page Web Applications, and it's important for developers to have the ability to ensure that the inter-dependencies within their own codebases and Third-Party modules are well defined and easily maintainable. This module is respnsible for the resolution of all relationships within a Web project leveraging WCM.
